Getting started

For those of you just starting out, I'd recommend getting the latest version of Anaconda. What is Anaconda? Essentially it's an easy way to download and install everything you'll need to get started performing data analysis in python. Unless you have a clear idea of what you're doing, just download Anaconda. While python 2.7 is still kicking around, python 3 is here to stay. For those starting out make sure you're downloading the python 3 version of Anaconda: Anaconda3. Most of the stability issues with Python 3 are in the past and moving forward Python 3 will likely be the standard for Python. Included in Anaconda:

  • Python with all of it's base libraries
  • pip and conda: how you'll install and update python libraries
