Peace Corps ESL Effectiveness Study

This is essentially one long Jupyter Notebook being used in the way Jupyter was intended for! As a flexible platform to both program and a detail research in one document. Here you can not only learn about the state of English Language education in Kosovo, but also see get a good demonstration of data mining techniques with Pandas and static graphing with Matplotlib and Seaborn.

Donor's Choose Grant Application Rater

This was a group project I did while at UCI. My group mates and I were interested in doing something with natural language processing along with doing creating ML tools that provide social goods. Our web app mirrors the application process of donorschoose.org a kickstarter-esque platform for teacher to sumbit grant proposals to the public. But instead our application provides users a prediction on how well their application is likely to get funded using NLP techniques.

Travel Informers

This is another group project completed at UCI, here we mainly aimed at experimenting with Javascript and thinking about how to make data useful to end users.
